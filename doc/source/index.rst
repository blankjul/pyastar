
pyastar - A* Algorithm in Python
====================================================================

You can find a documentation of pyastar at: https://www.egr.msu.edu/coinlab/pyastar/


Installation
====================================================================

The official release is always available at PyPi:

.. code:: bash

    pip install -U pyastar
